## Project Description
A To-Do List application implementing all CRUD (Create, Read, Update, Delete) operations using JavaScript and LocalStorage for persistent data storage.

## CRUD Operations Demonstrated

### CREATE
- Adding new tasks via input field
- Tasks saved with unique ID and timestamp
- Enter key support for quick addition

### READ
- Loading tasks from LocalStorage on startup
- Displaying all tasks in a clean interface
- Showing task statistics (total/completed/pending)

### UPDATE
- Toggling task completion with checkboxes
- Editing task text via prompt
- Automatic timestamp updates

### DELETE
- Removing individual tasks with confirmation
- Clearing all completed tasks at once
- Data persistence in browser storage

## How to Run the Application

### Method 1: Direct File Open

2. Double-click `index.html`
